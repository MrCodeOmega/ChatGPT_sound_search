Bu programı kullanmak için bilgisayarınız da python güncel sürümü yüklü olmalıdır.

Ardında herhangi bir python derleyicisi kullanarak (VsCode önerilir) 

Sırasıyla :
1) OpenAI sitesinden bir api key alıp "openai.api_key" değişkenine atamanız gerek.

2) pip install SpeechRecognition  (pip python modüle yükleme birimidir kullandığınız derleyicinin terminaline veya cmd ekranına yazmanız gereklidir.)

3) pip install PyAudio

4) pip install --upgrade openai

Bu kurulumların ardından kod bloğunun en sonunda bulunan "chatbot_voice_input()" fonksiyonunu veya tüm kod bloğunu çalıştırmanız yeterlidir.

Program çalıştığında terminalde "Say something!" yazısını gördükten sonra program sesinizi dinliyor olacaktır.

Artık aratmak istediğiniz kelimeyi söyleyerek chatgpt den geri dönüş alabilirsiniz.
